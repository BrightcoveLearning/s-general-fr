Limitations des ressources à

Les limitations suivantes doivent être prises en compte lors de l'utilisation de vidéos de ressources distantes:

  • SSAI n'est pas pris en charge pour les ressources distantes.
  • Les DRM ne sont pris en charge que si vous pouvez personnaliser le lecteur pour communiquer avec le serveur de licences DRM. C'est possible avec le lecteur Brightcove via un plugin. Pour les autres lecteur s, vous devrez consulter la documentation du lecteur.
  • Toutes les vidéos ingérées dans Video Cloud peuvent être lues dans le lecteur Brightcove. Pour les lecteurs autres que Brightcove, que vous rencontrerez principalement dans Brightcove Beacon, vous devrez vérifier que le format vidéo est entièrement pris en charge par chaque plate-forme. En outre, certaines fonctionnalités avancées de DASH et HLS ne sont pas encore prises en charge par Brightcove.

Pour plus d'informations sur les vidéos de ressources distantes, consultez le document Video Cloud Working with Remote Asset Videos. Sachez que vous allez visionner cette vidéo dans un nouvel onglet.
