Choose the languages for your apps

During the pre-sales process you chose to translate your apps into multiple languages. The possibilities for translation include the app fields and content metadata.

App translations are accomplished by requesting the translation of the app fields in your app experience. Content translations are the metadata that you enter into the CMS. It includes text data like titles and descriptions and localized images.

Defining the languages you wish to use in your Brightcove Beacon apps is part of the onboarding process. Past the onboarding process, if you choose to start using multiple languages, or wish to add languages for which translations exist, please contact your Brightcove support person. For language translations that do not yet exist for Brightcove Beacon, please contact Brightcove support.

What is translated?

There are two parts of the apps that are/can be translated, they are:

  • Translations of the field names, buttons and descriptions in the apps can be provided by you or can be translated by Brightcove for you.
  • Translations of items you define, for instance page names or video metadata, and images that are entered into Brightcove Beacon by you.

How your viewers choose a language for the app

There are three ways your viewers can choose the language to use in the app. They are:

  • Select the language in the Settings page
  • Set the language of the browser (if applicable)
  • Set the language for your device (if applicable)

Change the language in the app

Viewers can choose the language in the Brightcove Beacon app itself. The following shows the process the viewer would follow using the Settings page to change the app language from English to German:

Set the preferred language in the browser

If a viewer sets a preferred language in a browser, and that language is supported in the Brightcove Beacon app, the app will automatically use that language. The viewer would not have to set the language in the Settings standard page.

Note: If a language is set in the actual Brightcove Beacon app, that will override browser settings.

All popular browsers have a way to set the preferred language. As an example, in Chrome the setting is changed in the Advanced section of the Settings option:

Set the language on your device

If your viewers are using the Brightcove Beacon apps on mobile devices, they can change the device language. For instance, on an iPhone go to Settings → General → Language & Region

Note: If a language is set in the actual Brightcove Beacon app, that will override device settings.

Localizing Video Information

There are many items you can localize around videos. Details for each are provided here:

Define video metadata languages

The basic process for localizing video metadata is:

  • Configure the desired language in Beacons Studio's Admin module.
  • In the video properties page, select the language for which you want to enter localized metadata.
  • Enter the localized metadata.
For further information on using multi-lingual metadata see the Working with Multi-Lingual Metadata document.

The first step for using multi-lingual metadata is selecting the language in which you want metadata in Beacon Studio by navigating to Admin → Language Settings. You first add the language then save it, as shown here:

  1. Add the language
  2. Save the language

Video metadata

This task is performed in Beacon Studio. Go the the Media module then click the video title to which you wish to add multi-lingual metadata. To actually enter the metadata, navigate to Overview → Video Information then select the language and finally enter the metadata.

  1. Select the language
  2. Enter the metadata

Video images

This task is performed in Beacon Studio. Go the the Media module then click the video title to which you wish to add localized images. To actually supply the localized images, click the Images tab then select the language and finally upload the images.

Video subtitles/captions

This task is performed in Beacon Studio. Go the the Media module then click the video title to which you wish to add multi-lingual subtitles/captions. To actually supply the subtitles/captions, click the Video and Tracks tab, then click Edit, and finally complete the form.

For complete details see the Adding Captions to Videos document.
  1. Select the language
  2. Complete the form for the new subtitles/captions

Audio tracks

This task is performed in Beacon Studio. Go the the Media module then click the title video to which you wish to add audio tracks. To actually supply the audio tracks navigate to Video and Tracks → Audio Tracks section, click Edit then complete the form.

For complete details see the Adding Audio Tracks to Videos document.
  1. Upload your audio track using either File Upload or URL Upload.
  2. Select Language and Variant. Click Add Audio Track
  3. Click Save.

Pages

Pages need only to have their titles localized. In Beacon Studio, navigate to the Layout module and click the page name you wish to alter. Click the gear icon next to the page name and you can add localized page names.

Page Titles

Localizing Series information

Series can have their titles, images and metadata localized. These tasks are performed in Beacon Studio. Go the the Media module then click All series. Click the series title to open its properties page. Enter localized data by:

  1. Choose the language.
  2. Update the information. ONLY the data in the highlighted area can be localized.

Localizing Season information

Seasons can have their titles, images and metadata localized. These tasks are performed in Beacon Studio. Go the the Media module then click All series. Click the series title to which the season you wish to update belongs. Enter localized season data by:

  1. Choose the season to update.
  2. Choose the language.
  3. Update the season information. ONLY the data in the highlighted area can be localized.

Localizing Playlist Information

This task must be completed in Beacon Classic.

Playlists can only have their titles localized. In Beacon Classic UI, navigate to the Playlists section and click the playlist title you wish to alter. Go to the Textual Data tab to alter the titles.
